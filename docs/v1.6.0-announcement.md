# 🎉 PVE 多用户控制面板 v1.6.0 更新公告

## 📡 重磅更新：端口转发管理

本次版本的核心功能 —— **端口转发管理** 现已上线！支持全生命周期管理，自动同步 iKuai 软路由，实现网络配置一体化。

### 🌐 端口转发核心功能

| 功能 | 说明 |
|------|------|
| 添加端口转发 | 为已分配的 VM/LXC 快速创建端口转发规则，自动匹配内网 IP |
| 编辑 / 删除 | 修改端口号、协议或目标 IP，删除规则，实时同步 iKuai 路由器 |
| 启用 / 禁用 | 一键开关端口转发，无需登录路由器后台 |
| 端口范围限制 | 管理员可配置允许的端口范围，防止随意占用高危端口 |
| 用户端口限额 | 限制每个用户可创建的端口转发数量，避免资源滥用 |
| 自动同步 iKuai | 新增、修改、删除端口转发自动同步到 iKuai 路由器，实时生效 |
| 启动自动导入 | 服务重启时自动拉取 iKuai 已有规则，按 IP+端口匹配导入，不遗漏、不重复 |

### 🧹 级联清理

解绑或销毁 VM/LXC 时，自动清理关联的所有端口转发规则（本地数据库 + iKuai 路由器），无需人工干预。

## 🔗 DHCP 静态绑定

虚拟机自动获取固定 IP，告别 IP 漂移导致的网络中断。

### 🤖 自动分配

- VM/LXC **分配时自动创建** DHCP 静态绑定，IP 从指定范围随机选取空闲地址
- VM/LXC **解绑或销毁时自动删除** 静态绑定，释放 IP 资源
- 已存在 MAC 绑定的虚拟机自动跳过，返回已有 IP，不会重复创建

### ⚙️ 可配置

管理员可在 **管理后台 → 网络管理** 中配置：

| 配置项 | 说明 |
|--------|------|
| IP 分配范围 | 默认 `10.0.0.110` ~ `10.0.0.199`，可自由调整 |
| 所属 LAN 接口 | 下拉选择 iKuai 实际 LAN 接口，无需手动填写 |
| 网关 / DNS | 自定义 DHCP 分配的网关和 DNS 服务器 |

### 🔄 历史绑定同步

对于已在 iKuai 上手动添加过 DHCP 静态绑定的用户，支持**一键同步**：

1. 进入 **管理后台 → 网络管理 → DHCP 静态绑定设置**
2. 点击 **「🔄 从爱快同步」**
3. 系统自动匹配 `VM-{id}` / `CT-{id}` 格式的备注，回写数据库
4. 添加端口转发时直接读取数据库中的 IP，无需重复查询

## 🖥️ 接口列表优化

- 刷新接口时**同时获取 WAN 和 LAN** 接口，分别用于端口转发和 DHCP 配置
