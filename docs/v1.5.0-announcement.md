# 🎉 PVE 多用户控制面板 v1.5.0 更新公告

---

## 📦 重磅更新：LXC 容器管理

本次版本的核心功能 —— **LXC 容器全生命周期管理** 现已上线！全面覆盖容器创建、分配、操作、监控与维护：

### 🚀 容器基础操作

| 功能 | 说明 |
|------|------|
| 开机 / 关机 / 重启 / 强制停止 | 一键控制容器状态，操作需确认遮罩防误触 |
| 网页终端 | 通过 Xterm.js 网页终端直连容器控制台，无需额外工具 |
| 重置密码 | 一键重置容器 root 密码，无需进入宿主机操作 |
| 销毁容器 | 管理员可销毁容器，需输入 `yes` 确认，不可逆操作 |

### 📸 快照管理

- 创建 / 回滚 / 删除快照，支持多选、全选、批量删除
- **运行中的容器也可创建和回滚快照**，快照本身就是为快速备份和恢复而设计
- VM 快照同样解除关机限制，运行中的 VM 快照自动保存内存状态

### 💾 备份管理

- 创建备份、查看备份历史、恢复备份、删除备份
- 后台实时轮询备份/恢复进度，完成后自动通知
- 每日备份次数限制，防止磁盘被撑爆

### 🔄 其他操作

- 分配 / 换绑容器给其他用户
- CDK 兑换码续费容器
- 到期自动关机 + 站内信 / 邮件提醒

## 🎨 UI / UX 全面优化

### 按钮分组收纳

VM 和 LXC 卡片的操作按钮按功能重新组织：

| 分组 | 包含操作 |
|------|----------|
| 🔘 电源控制组 | `启动` `关机` `重启` `停止` — 连续按钮组，紧凑美观 |
| 🖥️ 独立展示 | VM 的 `VNC` / LXC 的 `终端` — 最常用操作，一眼可见 |
| 📋 管理菜单 | `快照` `备份` `重置密码` `编辑` `销毁` — 收纳下拉菜单，告别按钮墙 |

### LXC 卡片移除 VNC

LXC 容器不适用 VNC 控制台，已移除 LXC 卡片的 VNC 按钮，统一使用网页终端替代，更符合 LXC 管理习惯。

### 导航栏动态化

顶部导航栏不再硬编码，改为通过 API 动态渲染：

- **管理员**：`虚拟机管理` `LXC容器管理` `管理后台` `用户中心`
- **普通用户**：`我的虚拟机` `我的LXC容器` `用户中心`

后续如需增删导航项，只需修改服务端一处配置，所有页面自动同步。
